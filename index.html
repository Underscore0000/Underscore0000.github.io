About me

Available for collabs

My work focuses on building software for low-level penetration testing, ethical hacking, and offensive security research. I'm particularly interested in understanding how systems behave beneath their abstractions, whether that means interacting directly with operating system internals, network protocols, or native binaries. Most of my projects are designed to solve practical problems: developing tools that automate repetitive tasks, simplify security assessments, or provide a deeper understanding of how modern systems operate. I enjoy turning complex technical concepts into efficient, reliable software that security professionals can actually use.

GitHub serves as a living archive of that work. It contains open-source tools, experimental projects, and research-driven utilities that reflect my interest in systems programming, cybersecurity, and low-level software development. Every project is an opportunity to explore new techniques, refine engineering practices, and contribute practical solutions to the security community.
